## 1. 项目简介

### 1.1 项目背景

VNERP（越南大昌科技 ERP 系统）是专为丝网印刷行业设计的企业资源规划系统。系统针对丝网印刷行业的特殊需求，重点解决了以下行业痛点：

- **物料管理困难**：原材料以大包装采购，生产时需按小单位领用
- **批次追溯复杂**：无法精确追溯到每个小料单元的来源和去向
- **先进先出难执行**：传统管理方式难以严格执行 FIFO 原则
- **成本核算模糊**：无法精确核算每个工单、每个批次的实际成本
- **质量追溯困难**：出现质量问题时难以快速定位原因

### 1.2 系统定位

VNERP 是一套完整的 ERP 系统，覆盖企业核心业务流程：

```
销售订单 → 生产计划 → 物料采购 → 仓库管理 → 生产执行 → 品质检验 → 成品发货 → 财务核算
```

### 1.3 核心特色

| 特色 | 说明 |
|------|------|
| 小料拆分管理 | 支持整料自动拆分为小料，每个小料独立追溯 |
| FIFO 强制执行 | 系统强制执行先进先出，禁止非 FIFO 出库 |
| 二维码全流程追溯 | 从原材料到成品的完整追溯链 |
| 工单级成本核算 | 精确到工单、批次、小料的成本核算 |
| 标准卡管理 | 工艺参数、质量标准数字化管理 |

## 4. 核心业务流程

### 4.1 采购入库流程

```
采购订单 → 审核通过 → 生成整料二维码 → 扫码入库 → 自动拆分小料 → IQC检验
```

### 4.2 生产领料流程

```
工单下发 → 自动生成领料单 → FIFO推荐批次 → 扫码出库 → 投料记录
```

### 4.3 生产报工流程

```
开始报工 → 扫描工单二维码 → 查看标准卡 → 记录实际参数 → 品质检验 → 完工入库
```

### 4.4 销售发货流程

```
销售订单 → 审核通过 → 生成发货单 → FIFO推荐成品 → 扫码发货 → 生成应收单
```

## 5. 核心原则

### 5.1 先进先出 (FIFO) 原则

- 系统自动推荐最早入库的批次
- 禁止手动选择非推荐批次
- 异常领用需要审批
- 完整的操作日志记录

### 5.2 小料拆分原则

- 整料入库后必须拆分为小料
- 每个小料独立二维码
- 建立父子关联关系
- 余料单独管理，优先领用

### 5.3 业务驱动财务原则

- 所有财务数据来自业务单据
- 禁止手工录入财务凭证
- 实时生成财务数据
- 全流程可追溯
