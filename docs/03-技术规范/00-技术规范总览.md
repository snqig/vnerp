## 阅读约定

- 所有规范文档对应代码以 `src/` 目录为准；冲突时以代码现状为准。
- 涉及命令的，统一使用 `pnpm <script>` 形式（项目通过 `preinstall` 强制使用 pnpm）。
- 字段名/表名示例直接来自 `src/lib/db/schema.ts` 与各 API Route 中的 SQL。
- 文档维护原则：代码变更后同步修订，避免规范与实现脱节。
