# 丝网印刷行业基础知识

> 文档编号：VNERP-WIKI-BIZ-001 | 版本：V1.0 | 更新日期：2026-05-10

## 丝网印刷工艺概述

丝网印刷（Screen Printing）是一种使用丝网作为版基，通过感光制版方法制作带有图文的丝网印版，利用丝网印版的网孔漏印油墨的印刷方式。

### 核心工艺流程

```
原稿设计 → 制版（晒版） → 调墨 → 印刷 → 干燥/固化 → 检验 → 成品
```

## 关键物料

### 1. 油墨（Ink）

| 类型 | 说明 | 管理要点 |
|------|------|---------|
| 溶剂型油墨 | 传统油墨，含有机溶剂 | 需管理溶剂配比、挥发性 |
| 水性油墨 | 环保型，以水为溶剂 | 需管理干燥条件 |
| UV油墨 | 紫外线固化 | 需管理UV灯功率和固化时间 |

**VNERP 管理方式**：
- 油墨以大桶采购（如 20kg/桶）
- 生产时按标准卡配比调墨
- 剩余油墨需记录并追溯
- 通过 `prd_standard_card_ink` 记录配比

### 2. 丝网版（Screen）

| 参数 | 说明 | 管理要点 |
|------|------|---------|
| 目数 | 丝网每英寸的网孔数 | 目数影响印刷精度 |
| 张力 | 丝网绷紧程度 | 张力影响印刷质量 |
| 材质 | 不锈钢/聚酯/尼龙 | 不同材质适用不同场景 |

**VNERP 管理方式**：
- 通过 `prd_standard_card_screen` 记录网版参数
- 网版使用寿命管理
- 网版更换记录

### 3. 刀具/模切版（Die）

| 类型 | 说明 | 管理要点 |
|------|------|---------|
| 平压平模切 | 平面模切 | 适用于小批量 |
| 圆压圆模切 | 滚筒式模切 | 适用于大批量 |

**VNERP 管理方式**：
- 通过 `prd_standard_card_die` 记录刀具参数
- 刀具使用寿命管理
- 刀具更换记录

## 生产管理特点

### 1. 大包装拆分需求

丝网印刷行业的原材料通常以大包装采购，但生产时需要按小单位领用：

```
采购：1桶（20kg）→ 拆分为 4小桶（5kg）→ 领用 1小桶（5kg）
```

VNERP 通过 `qrcode_record.split_flag` 和 `parent_qr_id` 实现拆分追溯。

### 2. 批次追溯需求

每个印刷批次需要追溯到：
- 使用的油墨批次和配比
- 使用的网版编号和状态
- 使用的刀具编号
- 操作人员和报工记录

VNERP 通过标准卡 + 二维码追溯实现完整链路。

### 3. 成本核算需求

需要精确核算每个工单的：
- 油墨用量和成本
- 网版折旧成本
- 刀具折旧成本
- 人工成本
- 制造费用分摊
