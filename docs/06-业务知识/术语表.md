# 术语表

本表收录 印刷生产经营信息管理系统 Print MIS 印刷行业 ERP 系统中使用的业务术语与技术概念，按业务域分类。

## 1. 仓库与库存

| 术语 | 英文 | 说明 |
|---|---|---|
| 批次 | Batch | 同一批采购入库的物料，有独立批次号与二维码 |
| 整料 | Whole Material | 未拆分的原始物料，`split_flag=0` |
| 小料 | Split Material | 从整料拆分得到的物料，`split_flag=1` |
| 余料 | Remainder | 生产剩余回库的物料，`split_flag=2`，FIFO 优先消耗 |
| 可用数量 | Available Qty | 实际可分配的库存数量 |
| 预占数量 | Reserved Qty | 已审批未发料的数量 |
| FIFO | First In First Out | 先入先出分配算法，余料优先→近效期→先入 |
| 乐观锁 | Optimistic Lock | 基于 `version` 字段的并发控制机制 |
| 库存流水 | Inventory Transaction | 每笔库存变动的不可修改记录 |
| 盘点 | Stocktaking | 核对账面与实物库存差异 |
| 调拨 | Transfer | 跨仓库的库存转移 |
| 临期预警 | Expiry Warning | 接近过期日期的批次预警 |
| 呆滞物料 | Obsolete Material | 长期无变动的库存 |

## 2. 生产与工艺

| 术语 | 英文 | 说明 |
|---|---|---|
| 标准卡 | Process Card | 产品的工艺标准定义，含 BOM/工序/质量项目 |
| 样品卡 | Sample Card | 打样阶段的工艺卡，`card_type=sample` |
| 量产卡 | Mass Card | 量产阶段的工艺卡，`card_type=mass` |
| 工单 | Work Order | 生产任务单，关联产品与标准卡 |
| 物料领用 | Material Requisition | 生产向仓库领料的单据 |
| 超额领用 | Over Requisition | 超出标准 BOM 的领用，需审批 |
| 补充领用 | Supplementary Requisition | 生产中临时补充的领用 |
| 退料 | Return | 生产剩余物料退回仓库 |
| 工序 | Process | 生产流程的一个步骤 |
| 报工 | Production Report | 工序完成后上报工时与产量 |

## 3. 销售与发货

| 术语 | 英文 | 说明 |
|---|---|---|
| 销售订单 | Sales Order | 客户下达的订单 |
| 发货单 | Shipment | 从销售订单生成的发货单据 |
| 扫码发货 | Scan Ship | 通过扫描二维码完成发货出库 |
| 部分发货 | Partial Ship | 订单数量大于本次发货数量的发货 |
| 拣货 | Picking | 仓管员按发货单拣选物料 |
| 状态机 | State Machine | 发货单 11 状态的流转控制 |

## 4. 采购与财务

| 术语 | 英文 | 说明 |
|---|---|---|
| 采购订单 | Purchase Order | 向供应商下达的采购单 |
| 应收单 | Receivable | 发货后自动生成的应收款项 |
| 应付单 | Payable | 采购入库后自动生成的应付款项 |
| 收款 | Receipt | 客户付款记录 |
| 付款 | Payment | 向供应商付款记录 |
| 工单成本 | Work Order Cost | 工单归集的物料/人工/制造/外包成本 |
| 批次成本 | Batch Cost | 采购批次的单位成本，用于 FIFO 发料核算 |
| 业务驱动财务 | Business-Driven Finance | 财务单据由业务事件自动生成 |

## 5. 二维码与追溯

| 术语 | 英文 | 说明 |
|---|---|---|
| 二维码 | QR Code | 物料/产品/工单的唯一标识 |
| 父子关联 | Parent-Child | 拆分物料通过 `parent_qr_code` 关联 |
| 追溯 | Trace | 通过二维码查询全链路业务记录 |
| 扫码日志 | Scan Log | 每次扫码的行为记录 |
| 二维码类型 | QR Type | material/product/workorder/ink 等 9 种 |
