# v0.1.0 版本发布说明

> 发布日期：2026-05-10

## 新增功能

### 核心架构
- Next.js 15 + App Router 全栈架构
- MySQL 8.0 数据库
- JWT 认证与 RBAC 权限系统
- 统一 API 响应格式

### 生产管理
- 生产工单管理（创建、排产、派工、完工）
- 生产报工（印刷报工、模切报工、装配报工）
- 标准卡管理（油墨明细、网版明细、刀具明细、工序明细）

### 仓库管理
- 入库管理（采购入库、生产入库）
- 出库管理（领料出库、销售出库）
- 盘点管理（定期盘点、不定期盘点、循环盘点、抽盘）
- 调拨管理（库位调拨、仓库调拨）
- 二维码全流程追溯
- 物料自动拆分
- FIFO 先进先出分配

### 采购管理
- 采购申请
- 采购订单

### 品质管理
- 来料检验（IQC）
- 过程检验（IPQC）
- 成品检验（FQC）
- 出货检验（OQC）

### 销售管理
- 销售订单
- 发货管理
- 退货管理

### 财务管理
- 应收账款
- 应付账款
- 成本核算

### 印刷车间
- 油墨管理
- 网版管理
- 标签管理
- 二维码追溯

## 已知问题

- 大数据量查询性能待优化
- JWT Token 刷新机制待实现
