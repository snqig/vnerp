## 快速命令

```bash
pnpm dev              # 启动开发服务器（端口 5000）
pnpm dev:webpack      # webpack 模式（规避 Turbopack Windows globals.css 崩溃）
pnpm build            # 生产构建
pnpm ts-check         # TypeScript 类型检查
pnpm lint             # ESLint 检查
pnpm test:unit        # 单元测试（Vitest）
pnpm test             # E2E 测试（Playwright）
pnpm setup:db         # 全量初始化数据库（建库 + 导入 schema + 可选 seed）
pnpm migrate          # 增量迁移（up）
pnpm migrate:status   # 查看迁移状态
pnpm db:studio        # Drizzle Studio
pnpm backup           # 数据库备份
```

> 注意：`pnpm db:generate` / `db:push` / `db:migrate` 已废弃（仅打印警告），请改用 `pnpm setup:db`（全量初始化）或 `pnpm migrate`（增量迁移）。
