## 快速命令

```bash
pnpm dev              # 启动开发服务器（端口 5000）
pnpm build            # 生产构建
pnpm ts-check         # TypeScript 类型检查
pnpm lint             # ESLint 检查
pnpm test:unit        # 单元测试（Vitest）
pnpm test             # E2E 测试（Playwright）
pnpm db:studio        # Drizzle Studio
pnpm db:generate      # 生成迁移
pnpm db:migrate       # 执行迁移
```
