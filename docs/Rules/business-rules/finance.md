# 财务核算业务规则

> 文档编号：VNERP-BIZ-005 | 版本：V1.0 | 更新日期：2026-05-10

## 1. 应收账款

### 1.1 应收生成规则

| 来源 | 触发时机 | 金额计算 |
|------|---------|---------|
| 销售出库 | 销售发货确认 | 出库数量 × 销售单价 |
| 销售退货 | 退货入库确认 | -退货数量 × 销售单价（冲减） |

### 1.2 应收状态

```
1(未收款) → 2(部分收款) → 3(已结清)
```

### 1.3 应收数据表

使用 `fin_receivable` 表，核心字段：
- `receivable_no` - 应收单号
- `source_type` - 来源类型
- `source_no` - 来源单号
- `customer_id` / `customer_name` - 客户
- `amount` - 应收金额
- `received_amount` - 已收金额
- `balance` - 未收余额 = amount - received_amount
- `due_date` - 到期日
- `status` - 状态（1-3）

### 1.4 逾期规则

- 超过 `due_date` 未结清的应收自动标记为逾期
- 逾期 30 天以上标记为"关注"
- 逾期 90 天以上标记为"呆账"
- 逾期 180 天以上可申请坏账核销

## 2. 应付账款

### 2.1 应付生成规则

| 来源 | 触发时机 | 金额计算 |
|------|---------|---------|
| 采购入库 | 入库确认 | 入库数量 × 采购单价 |
| 采购退货 | 退货出库确认 | -退货数量 × 采购单价（冲减） |

### 2.2 应付状态

```
1(未付款) → 2(部分付款) → 3(已结清)
```

### 2.3 应付数据表

使用 `fin_payable` 表，核心字段：
- `payable_no` - 应付单号
- `source_type` - 来源类型
- `source_no` - 来源单号
- `supplier_id` / `supplier_name` - 供应商
- `amount` - 应付金额
- `paid_amount` - 已付金额
- `balance` - 未付余额 = amount - paid_amount
- `due_date` - 到期日
- `status` - 状态（1-3）

## 3. 成本核算

### 3.1 成本归集

```
工单成本 = 直接材料成本 + 直接人工成本 + 制造费用
```

| 成本项 | 数据来源 | 计算方式 |
|--------|---------|---------|
| 直接材料 | 领料出库记录 | 实际领料数量 × 物料单价 |
| 直接人工 | 报工记录 | 实际工时 × 工时费率 |
| 制造费用 | 费用分摊 | 按工时比例分摊 |

### 3.2 成本核算周期

- 实时核算：每次报工/领料时更新
- 期末核算：月末进行成本调整和差异分摊
- 成本差异：标准成本与实际成本的差异分析

### 3.3 利润计算

```
毛利润 = 销售收入 - 销售成本
毛利率 = 毛利润 / 销售收入 × 100%

销售成本 = 直接材料 + 直接人工 + 制造费用
```

## 4. 收款/付款记录

### 4.1 收款记录

- 每次收款需记录：收款金额、收款方式、收款日期、关联应收
- 收款后自动更新应收的 `received_amount` 和 `balance`
- `balance` 为 0 时自动更新状态为"已结清"

### 4.2 付款记录

- 每次付款需记录：付款金额、付款方式、付款日期、关联应付
- 付款后自动更新应付的 `paid_amount` 和 `balance`
- `balance` 为 0 时自动更新状态为"已结清"
