# 生产管理业务规则

> 文档编号：VNERP-BIZ-001 | 版本：V1.0 | 更新日期：2026-05-10

## 1. 工单管理

### 1.1 工单状态流转

```
1(待排产) → 2(已排产) → 3(生产中) → 4(已完工) → 5(已关闭)
                ↓             ↓
              5(已取消)     5(已取消)
```

规则：
- 待排产状态可修改工单信息
- 已排产状态可进行派工
- 生产中状态可进行报工
- 已完工状态可查看但不可修改
- 已取消状态不可恢复

### 1.2 工单编号规则

```
WO{年月日}{4位序号}
示例：WO202605100001
```

### 1.3 物料需求计算

```
物料需求数量 = 工单计划数量 × BOM单位用量 × (1 + 损耗率)
```

## 2. 报工管理

### 2.1 报工类型

| 类型 | 说明 | 关联 |
|------|------|------|
| 印刷报工 | 印刷工序完工 | 关联标准卡、网版、油墨 |
| 模切报工 | 模切工序完工 | 关联刀具模板 |
| 装配报工 | 装配工序完工 | - |

### 2.2 报工规则

- 报工数量不得超过工单计划数量
- 报工必须关联工单
- 报工必须记录操作人员
- 报工自动更新工单完工数量
- 不良品数量单独记录

### 2.3 报工数据表

使用 `prd_work_report` 表，核心字段：
- `work_order_id` - 关联工单
- `report_type` - 报工类型（1-印刷 2-模切 3-装配）
- `good_qty` - 良品数量
- `defect_qty` - 不良品数量
- `die_template_id` - 刀具模板ID（模切报工时必填）

## 3. 标准卡管理

### 3.1 标准卡结构

标准卡是丝网印刷的核心工艺文档，包含：

| 子表 | 说明 | 核心字段 |
|------|------|---------|
| prd_standard_card_ink | 油墨明细 | 油墨类型、颜色、配比、用量 |
| prd_standard_card_screen | 网版明细 | 网版类型、目数、张力 |
| prd_standard_card_die | 刀具明细 | 刀具模板、模切方式 |
| prd_standard_card_process | 工序明细 | 工序顺序、工序名称、标准工时 |

### 3.2 标准卡编号规则

```
SC{年月日}{4位序号}
示例：SC202605100001
```

### 3.3 标准卡使用规则

- 每个产品可关联多个标准卡（不同印刷方案）
- 工单必须关联标准卡
- 标准卡变更需创建新版本，旧版本标记为历史

## 4. 成本计算

### 4.1 直接材料成本

```
直接材料成本 = Σ(各物料实际用量 × 物料单价)
```

### 4.2 直接人工成本

```
直接人工成本 = Σ(各工序报工工时 × 工时费率)
```

### 4.3 制造费用分摊

```
制造费用分摊 = 工单直接工时 / 车间总工时 × 车间制造费用总额
```

### 4.4 工单总成本

```
工单总成本 = 直接材料成本 + 直接人工成本 + 制造费用分摊
```
