# 品质管理业务规则

> 文档编号：VNERP-BIZ-004 | 版本：V1.0 | 更新日期：2026-05-10

## 1. 检验类型

| 类型 | 说明 | 触发时机 | 检验对象 |
|------|------|---------|---------|
| 来料检验(IQC) | 采购物料入库检验 | 采购入库前 | 原材料 |
| 过程检验(IPQC) | 生产过程抽检 | 生产报工时 | 在制品 |
| 成品检验(FQC) | 成品入库检验 | 成品入库前 | 成品 |
| 出货检验(OQC) | 出货前检验 | 销售出库前 | 成品 |

## 2. 检验流程

### 2.1 来料检验(IQC)

```
创建检验单 → 抽样 → 检验 → 判定(合格/不合格/让步接收)
                              ↓合格            ↓不合格
                           入库           退货/特采
```

### 2.2 检验判定规则

| 判定 | 条件 | 后续处理 |
|------|------|---------|
| 合格 | 不良率 < AQL标准 | 正常入库 |
| 不合格 | 不良率 >= AQL标准 | 退货或特采审批 |
| 让步接收 | 不良率在临界值 | 需主管审批 |

### 2.3 AQL标准

| 检验水平 | 抽样比例 | 接收质量限 |
|---------|---------|-----------|
| 一般检验I | 8% | 1.0% |
| 一般检验II | 13% | 1.5% |
| 加严检验 | 20% | 0.65% |

## 3. 不合格品处理

### 3.1 处理方式

| 方式 | 说明 | 审批级别 |
|------|------|---------|
| 退货 | 退回供应商 | 品质主管 |
| 特采 | 让步接收使用 | 生产经理 |
| 返工 | 返工后重新检验 | 品质主管 |
| 报废 | 直接报废处理 | 总经理 |

### 3.2 不合格品追溯

- 记录不合格品对应的物料批次
- 关联采购订单和供应商
- 记录不良现象和不良原因
- 8D报告关联

## 4. SPC统计过程控制

### 4.1 控制图类型

| 图表 | 适用场景 | 监控对象 |
|------|---------|---------|
| X-bar R | 计量值数据 | 尺寸、重量 |
| P图 | 计数值数据 | 不良率 |
| C图 | 缺陷数 | 缺陷数量 |

### 4.2 控制限计算

```
UCL = X̄ + 3σ
CL = X̄
LCL = X̄ - 3σ
```

超出控制限自动触发异常预警。
