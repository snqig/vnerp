## 每日检查项

### 系统健康检查

| 检查项 | 方法 | 正常标准 |
|--------|------|---------|
| 服务状态 | `pm2 status` | 状态为 online |
| 磁盘空间 | `df -h` | 使用率 < 80% |
| 内存使用 | `free -h` | 使用率 < 85% |
| 数据库连接 | `mysqladmin ping` | 响应正常 |
| 错误日志 | `pm2 logs vnerp --err --lines 50` | 无异常错误 |

### 业务数据检查

| 检查项 | 方法 | 异常处理 |
|--------|------|---------|
| 库存预警 | 系统通知 | 生成采购申请 |
| 应收逾期 | 财务模块查看 | 催收跟进 |
| 待审批单据 | 系统通知 | 及时审批 |
| 生产异常 | 生产模块查看 | 协调处理 |

## 常用操作

### 采购入库操作流程

1. 采购员创建采购订单 → 确认
2. 仓库收到物料 → 创建来料检验单
3. 品质检验合格 → 创建入库单
4. 入库确认 → 库存增加 + 二维码生成 + 应付生成

### 生产领料操作流程

1. 生产人员创建领料单（关联工单）
2. 仓库确认出库 → FIFO 分配 → 二维码状态更新
3. 领料完成 → 工单物料消耗记录

### 销售发货操作流程

1. 销售员创建发货单（关联销售订单）
2. 仓库确认出库 → 库存扣减
3. 发货确认 → 应收生成
