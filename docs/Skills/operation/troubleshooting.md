# 常见问题排查 SOP

> SOP 编号：VNERP-SKILL-011 | 版本：V1.0 | 更新日期：2026-05-10

## 问题分类与排查

### 1. 页面无法访问

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 检查服务状态 | `pm2 status` |
| 2 | 检查端口监听 | `netstat -tlnp \| grep 5000` |
| 3 | 检查 Nginx 状态 | `systemctl status nginx` |
| 4 | 检查错误日志 | `pm2 logs vnerp --err --lines 100` |
| 5 | 重启服务 | `pm2 restart vnerp` |

### 2. API 返回 500 错误

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 查看服务端日志 | `pm2 logs vnerp --lines 200` |
| 2 | 检查数据库连接 | `mysql -u root -p -e "SELECT 1"` |
| 3 | 检查 SQL 错误 | 查看日志中的 SQL 语句和错误信息 |
| 4 | 检查磁盘空间 | `df -h` |
| 5 | 检查内存 | `free -h` |

### 3. 数据库连接失败

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 检查 MySQL 状态 | `systemctl status mysql` |
| 2 | 检查连接数 | `mysql -e "SHOW PROCESSLIST"` |
| 3 | 检查最大连接数 | `mysql -e "SHOW VARIABLES LIKE 'max_connections'"` |
| 4 | 检查 .env 配置 | 确认 DB_HOST/PORT/USER/PASSWORD |
| 5 | 重启 MySQL | `systemctl restart mysql` |

### 4. 页面加载缓慢

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 检查网络延迟 | `ping erp.yourcompany.com` |
| 2 | 检查服务器负载 | `top` / `htop` |
| 3 | 检查数据库慢查询 | `mysql -e "SHOW FULL PROCESSLIST"` |
| 4 | 检查 API 响应时间 | 浏览器 Network 面板 |
| 5 | 检查缓存命中率 | Redis 监控（如使用） |

### 5. 登录失败

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 检查用户名密码 | 确认输入正确 |
| 2 | 检查账户锁定 | 查看登录日志 |
| 3 | 检查 JWT 配置 | 确认 JWT_SECRET 一致 |
| 4 | 检查 Cookie 设置 | 浏览器开发者工具 → Application → Cookies |
| 5 | 重置密码 | 使用 reset-password.ts 脚本 |

### 6. 数据不一致

| 步骤 | 操作 | 命令/方法 |
|------|------|----------|
| 1 | 确认问题范围 | 哪些表、哪些数据 |
| 2 | 检查事务日志 | 查看是否有事务未提交 |
| 3 | 检查并发操作 | 是否有同时操作同一数据 |
| 4 | 数据修复 | 编写修复 SQL，在事务中执行 |
| 5 | 验证修复 | 确认数据一致性 |

## 紧急处理流程

```
发现问题 → 评估影响 → 通知相关人员 → 定位原因 → 实施修复 → 验证结果 → 记录总结
```
