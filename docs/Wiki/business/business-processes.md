# 核心业务流程图

> 文档编号：VNERP-WIKI-BIZ-002 | 版本：V1.0 | 更新日期：2026-05-10

## 1. 采购到付款流程（P2P）

```
需求产生 → 采购申请 → 审批 → 采购订单 → 供应商确认
    ↓                                        ↓
库存预警                                    发货
    ↓                                        ↓
MRP运算                                   到货验收
                                              ↓
                                         来料检验(IQC)
                                          ↓合格    ↓不合格
                                      采购入库    退货/特采
                                          ↓
                                    应付账款生成 → 付款 → 结清
```

## 2. 订单到收款流程（O2C）

```
客户询价 → 销售报价 → 销售订单 → 生产计划 → 生产执行
                                              ↓
                                         成品检验(FQC)
                                              ↓合格
                                         成品入库
                                              ↓
                                         出货检验(OQC)
                                              ↓
                                         销售发货 → 应收生成 → 收款 → 结清
```

## 3. 生产执行流程

```
工单下达 → 关联标准卡 → 物料需求计算 → 领料出库(FIFO)
    ↓                                        ↓
排产/派工                               物料拆分（如需）
    ↓                                        ↓
印刷报工 → 模切报工 → 装配报工 → 完工确认
    ↓           ↓          ↓
成本归集    二维码追溯   质量记录
```

## 4. 仓库管理流程

```
入库流程：
采购入库 → IQC检验 → 入库确认 → 生成二维码 → 自动拆分(如需) → 库存增加

出库流程：
领料申请 → FIFO分配 → 出库确认 → 二维码状态更新 → 库存减少

盘点流程：
创建盘点单 → 执行盘点 → 差异计算 → 审核确认 → 库存调整

调拨流程：
创建调拨单 → 审核 → 源仓库出库 → 目标仓库入库 → 库存更新
```

## 5. 二维码追溯流程

```
原材料入库 → 生成原料码(qrcode_record, qr_type=material)
    ↓
物料拆分 → 生成小料码(split_flag=1) + 余料码(split_flag=2)
    ↓                                        ↓
parent_qr_id 关联                        parent_qr_id 关联
    ↓
领料出库 → 更新状态(status=2) → 关联工单(ref_id)
    ↓
生产报工 → 记录使用的物料二维码
    ↓
成品入库 → 生成成品码(qrcode_record, qr_type=product)
    ↓
销售出库 → 更新状态(status=2)
```

## 6. 财务核算流程

```
采购入库 → 应付账款(fin_payable, status=1)
    ↓
付款 → 更新已付金额 → status=2(部分) 或 3(结清)

销售出库 → 应收账款(fin_receivable, status=1)
    ↓
收款 → 更新已收金额 → status=2(部分) 或 3(结清)

生产完工 → 成本归集
    ↓
直接材料(领料记录) + 直接人工(报工记录) + 制造费用(分摊)
    ↓
工单成本 = Σ(材料+人工+制造费用)
```
