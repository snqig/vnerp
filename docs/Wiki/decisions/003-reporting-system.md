# 技术决策记录：报表系统选型

> 编号：VNERP-DEC-003 | 日期：2026-05-10 | 状态：已决定

## 背景

ERP 系统需要提供各类报表功能，包括生产报表、库存报表、财务报表等。

## 选项

### 选项 A：前端图表组件（Recharts / ECharts）

- 优势：交互性好、无需后端渲染、开发快
- 劣势：大数据量时性能受限

### 选项 B：服务端报表引擎（JasperReports / BIRT）

- 优势：功能强大、支持复杂报表、可导出 PDF
- 劣势：Java 生态、部署复杂、学习成本高

### 选项 C：混合方案（前端图表 + CSV/Excel 导出）

- 优势：灵活、前端交互好、导出满足需求
- 劣势：复杂报表需自行实现

## 决策

选择 **选项 C：混合方案**。

## 理由

1. 前端图表满足日常查看需求
2. CSV/Excel 导出满足数据交换需求
3. 无需引入 Java 依赖
4. 开发效率最高
5. 后续可按需引入专业报表工具

## 影响

- 前端使用 Recharts 或 ECharts 渲染图表
- 后端提供 CSV/Excel 导出 API
- 复杂报表逻辑在前端实现
