## 1. 登录模块测试用例

### TC-LOGIN-001: 正常登录流程
| 项目 | 内容 |
|------|------|
| **用例编号** | TC-LOGIN-001 |
| **用例标题** | 使用有效凭据正常登录 |
| **测试目的** | 验证用户可以使用正确的用户名和密码登录系统 |
| **前置条件** | 1. 系统已部署并运行<br>2. 用户admin已存在且密码为admin<br>3. 网络连接正常 |
| **测试步骤** | 1. 访问登录页面 http://localhost:5000/login<br>2. 在用户名输入框输入"admin"<br>3. 在密码输入框输入"admin"<br>4. 点击"登录"按钮 |
| **预期结果** | 1. 页面跳转到系统首页<br>2. 显示用户头像和名称<br>3. 侧边栏显示完整菜单 |
| **优先级** | P1 - 高 |
| **测试类型** | 功能测试、冒烟测试 |
