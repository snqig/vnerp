## 1. 环境准备

### 1.1 系统要求

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 4核 | 8核及以上 |
| 内存 | 8GB | 16GB及以上 |
| 硬盘 | 100GB SSD | 200GB SSD及以上 |
| 操作系统 | Windows 10 / Server 2019 | Windows 11 / Server 2022 |
| Node.js | 18.x | 20.x LTS |
| MySQL | 8.0 | 8.0 |
| PNPM | 8.x | 9.x |

### 1.2 安装依赖

```bash
# 安装 Node.js (推荐使用 nvm)
nvm install 20
nvm use 20

# 安装 PNPM
npm install -g pnpm

# 安装 MySQL
# 下载地址：https://dev.mysql.com/downloads/mysql/

# 验证安装
node -v  # v20.x.x
pnpm -v  # 9.x.x
mysql --version  # 8.0.x
```

## 2. 数据库初始化

### 2.1 创建数据库

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE IF NOT EXISTS vnerpdacahng
  DEFAULT CHARACTER SET utf8mb4
  DEFAULT COLLATE utf8mb4_0900_ai_ci;

-- 创建专用用户（可选）
CREATE USER 'vnerp_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON vnerpdacahng.* TO 'vnerp_user'@'localhost';
FLUSH PRIVILEGES;
```

### 2.2 导入数据库脚本

```bash
# 进入项目目录
cd E:\vnerp-main\vnerp-main

# 导入主数据库结构
mysql -u root -p vnerpdacahng < database/vnerpdacahng_schema.sql

# 导入仓库相关表
mysql -u root -p vnerpdacahng < database/warehouse.sql
mysql -u root -p vnerpdacahng < database/warehouse_inbound.sql
mysql -u root -p vnerpdacahng < database/inventory_tables.sql

# 导入权限系统
mysql -u root -p vnerpdacahng < database/permission_system.sql

# 导入菜单数据
mysql -u root -p vnerpdacahng < database/dcprint_menus.sql

# 导入品质管理相关
mysql -u root -p vnerpdacahng < database/quality_menus.sql
mysql -u root -p vnerpdacahng < database/quality_incoming.sql

# 导入其他必要脚本
mysql -u root -p vnerpdacahng < database/organization.sql
mysql -u root -p vnerpdacahng < database/hr_menus.sql
```

### 2.3 验证数据库

```sql
-- 登录 MySQL
mysql -u root -p vnerpdacahng

-- 查看表列表
SHOW TABLES;

-- 验证核心表
SELECT COUNT(*) FROM sys_user;
SELECT COUNT(*) FROM sys_menu;
SELECT COUNT(*) FROM sys_role;
```

## 3. 项目配置

### 3.1 环境变量配置

```bash
# 复制环境变量模板
copy .env.example .env

# 编辑 .env 文件，配置以下参数：
```

**.env 文件示例：**

```env
# 数据库配置
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=Snqig521223
DB_NAME=vnerpdacahng

# 应用配置
NEXT_PUBLIC_API_URL=http://localhost:5000
PORT=5000

# JWT 密钥
JWT_SECRET=your-secret-key-here
JWT_EXPIRES_IN=7d
```

### 3.2 安装项目依赖

```bash
# 使用 PNPM 安装依赖
pnpm install

# 如果安装失败，尝试清除缓存后重新安装
pnpm store prune
pnpm install
```

### 3.3 构建项目

```bash
# 开发模式
pnpm dev

# 生产构建
pnpm build

# 启动生产服务
pnpm start
```

## 6. 常见问题

### 6.1 数据库连接失败

**问题现象：** 启动时报数据库连接错误

**解决方案：**
```bash
# 检查 MySQL 服务是否启动
net start mysql

# 检查数据库配置是否正确
cat .env | grep DB_

# 测试数据库连接
mysql -u root -p -e "SELECT 1"
```

### 6.2 依赖安装失败

**问题现象：** `pnpm install` 报错

**解决方案：**
```bash
# 清除缓存
pnpm store prune

# 删除 node_modules
rmdir /s /q node_modules

# 重新安装
pnpm install
```

### 6.3 构建失败

**问题现象：** `pnpm build` 报错

**解决方案：**
```bash
# 类型检查
pnpm ts-check

# 修复 ESLint 错误
pnpm lint:fix

# 重新构建
pnpm build
```

```bash

# 仅备份数据（不包含结构）
mysqldump -u root -p --no-create-info vnerpdacahng > backup_data_$(date +%Y%m%d).sql
```
